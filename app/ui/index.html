Rename by mask

Placeholders: %artist% %title% %album% %albumartist% %track% %disc% %year% %genre% %composer% %publisher% %bpm% %isrc% %key% %catalognumber%. Wrap optional parts in [brackets] — dropped whole when empty, e.g. %album%[ (%year%)]. Track numbers pad to two digits; set another width with %disc:2%.

Reorganize into folders

Use / (or \) to separate folders. Optional parts go in [brackets] and vanish when empty. Paths are built under the opened library; tag values can't introduce a folder, only separators in the pattern do.
